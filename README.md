# FUTURE_UX_02-new-ui-ux
🎨 Art Therapy Mobile App UI

📌 Overview
This project is a mobile app UI design for an **Art Therapy Appointment Booking System**, focused on improving mental well-being through creative sessions like painting, sketching, and guided therapy.

✨ Features
- Onboarding & Login Flow
- Therapy Session Exploration
- Appointment Booking Interface
- Confirmation Screen
- User Profile & Session Tracking

 🎯 UX Focus
- Simple and intuitive booking flow
- Emotion-driven design (calming & creative UI)
- Clean layout with strong visual hierarchy

 🛠 Tools Used
- Figma
